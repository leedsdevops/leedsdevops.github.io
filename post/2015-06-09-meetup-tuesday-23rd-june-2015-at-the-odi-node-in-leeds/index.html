Meetup - Tuesday 23rd June 2015 at The ODI Node in Leeds

Tuesday 23rd June 2015 sees us return to the Open Data Institute Leeds Node. As always we'll have two great speakers.

In Meetups

Tuesday 23rd June 2015 sees us return to the Open Data Institute Leeds Node. As always we’ll have two great speakers…

  • 6:45 - 7:15 : Grab a beer, have a natter.
  • 7:15 - 7:30 : Group updates/news/announcements
  • 7:30 - 8:00 : Security is Everyone’s Responsibility
  • 8:00 - 8:15 : Intermission
  • 8:15 - 9:00 : Docker: Please contain your excitement
  • 9:00 - late : Decamp to the Wardrobe

At our last meetup we asked the attendees to vote on the openspace discussion topics from Devopsdays Paris, there was a clear interest in security, we are nothing if not accommodating and start the meetup with a cracking talk “Security is Everyone’s Responsibility” from James Hall of Parallax. James will be talking about the state of website security today, showing off common flaws and explaining how we can all improve security for our customers.

For tonights main event we are super pleased to be announcing Mr John Leach will be taking a deep dive into docker with “Docker: Please contain your excitement”. It’s a look at the underlying technology of Docker, with an overview of Linux containerisation, anatomy of a Docker image, some security considerations and the wider Docker ecosystem.

Yet again we thank Brightbox for sponsoring our the evening, so there will be a generously stocked bar.

Venue

The ODI Node Leeds  is situated at Munro house, on Duke Street (not too far from the Adelphi). If you want to find out more about the ODI the details can be found at http://theodi.org/nodes/leeds
